# Prompt Evaluations

Python notebooks for prompt evaluation techniques using Claude (Anthropic) and NVIDIA-hosted models.

## Setup

### 1. Install dependencies

```bash
pip install anthropic openai python-dotenv jupyterlab
```

### 2. Configure API keys

Create a `.env` file in the project root:

```
ANTHROPIC_API_KEY=your-anthropic-api-key
NVIDIA_API_KEY=your-nvidia-api-key
```

The notebooks load this file automatically via `python-dotenv`. Never commit `.env` to version control.

### 3. Run Jupyter Lab

```bash
jupyter lab
```

Open the browser at `http://localhost:8888` and navigate to the notebook you want to run.

## Prompt Evaluation Process

Each evaluation notebook follows the same pipeline:

1. **Generate dataset** — the model produces a set of AWS-related tasks (Python, JSON, or Regex format) and saves them to a JSON file.
2. **Run prompts** — each task is sent to the model; the response is captured.
3. **Grade syntax** — the response is validated structurally (`ast.parse` for Python, `json.loads` for JSON, `re.compile` for Regex). Score: 0 or 10.
4. **Grade by model** — the same model acts as judge and scores the response 1–10 based on task-specific criteria.
5. **Compute final score** — average of the syntax score and model score.
6. **Print results** — full JSON output with scores and reasoning per test case.
